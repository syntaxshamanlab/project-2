This repository contains forensic analysis, psychological warfare manuals, and personal narratives related to the RyanrealAF project.

## Deployment

This site is ready for deployment on GitHub Pages.
- `index.html`: Main entry point listing all documents.
- `.nojekyll`: Ensures raw text files are served correctly without Jekyll processing.
- `notebooklm-import-raw/`: Contains the source text files.
