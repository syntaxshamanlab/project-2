# RyanrealAF Doctrine Archive

A static website showcasing the RyanrealAF master doctrine archive, featuring typography-focused UI design and comprehensive documentation on psychological warfare, counter-intelligence, and personal narratives.

## Features

- **Typography-Focused Design**: Clean, modern typography using the Inter font family
- **Responsive Layout**: Mobile-friendly design that works on all devices
- **Interactive Content**: Collapsible document sections with full-text viewing
- **Infographic Integration**: Displays the typography focus infographic prominently
- **Static Site**: Ready for deployment to GitHub Pages

## Content

The site showcases a collection of documents covering:
- Core doctrine and foundational theory
- Psychological warfare tactics and analysis
- Personal narratives and case studies
- Counter-intelligence strategies
- Systemic critique and recovery community vision

## Deployment to GitHub Pages

1. Push this repository to GitHub
2. Go to repository Settings > Pages
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Click Save

The site will be available at `https://yourusername.github.io/repository-name/`

## Local Development

To run locally:

```bash
npx http-server
```

Then open `http://localhost:8080` in your browser.
